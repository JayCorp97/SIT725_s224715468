## **Features**

- Add a new species with:
  - Name
  - Location
  - Confidence score
  - Image URL
- View all species in **responsive Bootstrap cards**
- Edit species **inline directly inside the card**
- Delete species with a single click
- Fully responsive and mobile-friendly
- Connected to a **MongoDB database** for persistent storage

## **Tech Stack**

- **Frontend:** HTML, Bootstrap 5, JavaScript  
- **Backend:** Node.js, Express  
- **Database:** MongoDB (local or Atlas)  
- **Version Control:** GitHub
